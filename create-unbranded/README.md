# create-unbranded

A launcher for [unbranded](https://www.npmjs.com/package/unbranded). Any of these drops you into the same interactive CLI:

```bash
npm create unbranded@latest
pnpm create unbranded
bun create unbranded
```

Every argument passes straight through, so `npm create unbranded -- --config recipe.json` behaves exactly like `unbranded --config recipe.json`.

This package is version-locked to `unbranded` and installs it as a dependency. See that package for what the tool does and how to configure it.
